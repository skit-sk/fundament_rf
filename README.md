# Fundament RF — Трекер сделок и визуализация

Веб-приложение для учёта торговых сделок с графиками отклонения цены от входа. Данные загружаются с биржи Bitget.

## Структура проекта

```
fundament_rf/
├── app.py                    # Точка входа Flask
├── models.py                # Модель данных FundObj
├── storage.py               # JSON-хранилище
├── flask_runner.py          # Автоперезапуск
├── routes/
│   ├── __init__.py
│   ├── api.py               # API: создание/удаление объектов, парсинг эмодзи
│   ├── web.py              # Веб-страницы: главная, карточка
│   └── graphics.py        # Графики: расчёт отклонений, max/min
├── templates/
│   ├── base.html          # Базовый шаблон с навигацией
│   ├── index.html         # Главная страница
│   ├── card.html          # Карточка сделки + график
│   └── graphics/
│       └── index.html     # Раздел графиков
├── static/css/style.css
└── data/*.json            # Данные карточек
```

## Карта API

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Главная страница |
| GET | `/card/<id>` | Карточка сделки |
| GET | `/graphics` | Раздел графиков |
| GET | `/graphics/chart/<id>` | JSON: исторические данные для графика |
| POST | `/api/objects/from-emoji` | Создание карточки из эмодзи-строки |
| DELETE | `/api/objects/<id>` | Удаление карточки |

## Формат входных данных (эмодзи-строка)

```
🏗️ [number] 🚏[symbol] 🧾[entry_price] 📆[entry_date] 🕒[days] 🧱[volume] 📈[pnl_percent] 🫧[pnl_usdt] 📦[result]
```

| Эмодзи | Поле | Описание |
|--------|------|----------|
| 🏗️ | number | Номер сделки |
| 🚏 | symbol | Торговая пара (FIL, ATOM, etc.) |
| 🧾 | entry_price | Цена входа |
| 📆 | entry_date | Дата входа (YYYY-MM-DD) |
| 🕒 | entry_time | Дней в сделке |
| 🧱 | volume | Объём |
| 📈/📉 | pnl_percent | Профит/убыток в % |
| 🫧 | pnl_usdt | Профит/убыток в USDT |
| 📦 | result | 🟢 (профит) / 🔴 (убыток) |

## Визуализация

### Карточка сделки

- Эмодзи-строка (краткая сводка)
- Таблица данных (3 строки: эмодзи → название → значение)
- График отклонения цены:
  - Ось X: дата
  - Ось Y: отклонение в % от цены входа
  - Линия entry price (фиолетовая пунктирная на 0%)
  - Зелёный/красный в зависимости от профит/убыток
- Под графиком: Mx (max) / Mn (min) за период

### Главная страница

- Список карточек с центрированным названием
- Эмодзи-строка для каждой карточки
- Max/Min отклонения (после первого открытия карточки)

### Раздел графиков

- Общий график выбранной сделки
- Статистика: период, количество точек, Mx/Mn

## Запуск

```bash
source venv/bin/activate
python app.py
# или через flask_manager:
python ../flask_manager.py start fundament_rf
```
